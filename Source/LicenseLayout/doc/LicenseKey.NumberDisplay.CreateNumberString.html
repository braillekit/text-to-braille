NumberDisplay.CreateNumberString Method 

Create a string representing the number.

public static string CreateNumberString(
   uint sum,
   int numberLength,
   bool useBase10
);

Parameters

sum
The number that you want the display for.
numberLength
The length of the display (visible) that you want.
useBase10
The base of the number you want, (10)true or (16)false.

Return Value

The resulting string.

Remarks

Remember that the length should be the visible part. This has nothing to do with bytes or bits.

Example

txtChkNumStr.Text = NumberDisplay.CreateNumberString(csum, numberLength, useBase10);
