易點雙視 v3.15.0

作業環境需求

  - Windows 7 SP1、Windows 8、Windows 10 （注意 Windows XP 已經不支援!）
  - .NET Framework 3.5 和 .NET Framework 4.7.2 都必須安裝。
  
安裝程序

1.確保已經安裝了所有最新的 Windows Updates。

2.安裝 .NET Framework 3.5 可轉散發套件。
  可至微軟網站下載，網址如下：  
  https://www.microsoft.com/zh-tw/download/details.aspx?id=21

  下載後直接開啟檔案即可安裝。

3.安裝 .NET Framework 4.7.2 可轉散發套件。
  可至微軟網站下載，網址如下：  
  http://go.microsoft.com/fwlink/?linkid=863265
  
  下載後直接開啟檔案即可安裝。
  
4.安裝「易點雙視」

  執行安裝程式 setup.exe 即可開始安裝。
  預設的應用程式安裝路徑為 C:\EasyBrailleEdit3。
