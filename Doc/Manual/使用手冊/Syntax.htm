原書頁碼

用途

標示原書頁碼。

語法

<P>n</P>

其中 n 為數字。

說明

一旦有標示原書頁碼，就必須完整標示。例如，您的文件一共有五頁，如果您的原書頁碼只標示到第 2 頁，那麼後面三頁的原書頁碼都會是第 3 頁。

頁標題

用途

標示原書頁碼。

語法

<標題>這裡放標題文字</標題>

說明

一份文件可以指定多個頁標題，因此您可以將整本書編輯成一個檔案，並且在各章開頭的地方加上<標題>。頁標題會印在頁尾的行首。

範例

<標題>第六課：自然的奧秘</標題>

縮排

用途

將內文整段縮排。

語法

<縮排>abc</縮排>

說明

被縮排標籤包住的文字將會內縮一個空方。您可以使用巢狀縮排標籤，例如：

<縮排>
1 Hello!
<縮排>
1.1 這裡會內縮兩個空方。
</縮排>
</縮排>

"1.1" 開頭的那一列，由於是包在第二個縮排區塊裡面，因此會內縮兩個空方。

座標

用途

表示座標。

語法

<座標>(x,y)</座標>

說明

括號裡的數字都不用加數符。

次方

用途

表示數學的次方。

語法

x^y 表示 x 的 y 次方。次方符號 '^' 的按鍵輸入方式為 Shift+6。

其中代表次方的數字 y 無需加數符。

範例

54m^2
