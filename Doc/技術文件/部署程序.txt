部署程序

1. 確保 ChangeLog.txt 已經編寫完成，而且所有改變皆已 commit，並且合併至 main branch。

2. 在 main branch 建立 git tag，例如："4.1.0"，並且推送至 git server。

   指令範例：git tag 4.1.0

3. 在專案原始碼的根目錄下執行命令：

   flubu compile
  
4. 執行 output 資料夾底下的 combine.bat。

5. 把 Combined\Release\ 資料夾底下的檔案手動揀選需要的檔案，並複製到 UpdateFiles 資料夾。

6. 到 UpdateFiles 資料夾下修改 Update.txt 內容，確保版號正確。

5.把 UpdateFiles 資料夾下的檔案複製到  https://github.com/huanlin/easy-braille-edit.git
  的 "Update4" 分支的 UpdateFiles 資料夾。  
  repo URL:
  https://github.com/huanlin/easy-braille-edit/tree/Update4

6.在本機測試能否自動更新。

7. 使用 InnoSetup 製作安裝程式：Setup\InnoSetup\iss。

注意：易點雙視無法在 Windows 11 Sandbox 環境中正常執行，應用程式載入時會發生錯誤：
     「找不到指定的模組。」
     但是在一般 Windows 10 或 Windows 11 環境上執行則沒有問題。
     參見：https://www.huanlintalk.com/2022/02/scintillanet-load-failed.html
